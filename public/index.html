Guide

产品使用说明

这个产品是做什么的

这是一个给 Polymarket 马斯克发帖次数市场做半自动分析的交易面板。它不会替你自动实盘下单， 重点是帮你快速看清：现在市场怎么报价、模型觉得哪些区间更合理、组合建仓后大概值不值得做。

最简单的使用顺序

  1. 先在左侧选一个市场，确认周期、成交额、流动性和剩余时间。
  2. 看 `xtracker count`，这是当前抓到的马斯克发帖计数；如果自动抓取失败，可以手动填。
  3. 输入你的 `预算金额` 和 `安全边际`，再选择一种 `建仓策略`。
  4. 在表格里对比 `市场概率`、`模型概率` 和 `Edge`，勾选你准备关注的几个区间。
  5. 看右侧的 `组合建仓模拟`，决定是直接建仓、挂单等待，还是先观察。

核心字段怎么理解

xtracker count

当前已统计到的马斯克发帖数，是模型判断最重要的实时输入之一。

市场概率

市场当前报价隐含出来的概率，代表别人现在大概愿意按什么价格交易。

模型概率

系统结合当前计数、时间进度和区间分布后，算出来的“你这边的判断”。

Edge

简单理解就是“模型觉得值不值”减去“市场要你付出的价格”再减去安全边际，越高越值得看。

组合成本

你勾选的几个区间一起买入时，大概需要花掉多少成本。

命中概率

你勾选的这些区间加起来，被模型命中的总概率。

两种建仓策略的区别

Settlement Edge

偏结算思路。它更关注“如果我一直拿到结算，这笔组合整体值不值”，适合想做区间覆盖和结算收益的人。

Convergence Edge

偏波段思路。它更关注“市场价格会不会往模型判断靠拢”，适合做挂单、等回落、做中间波动的人。

右侧建议怎么用

结算建议

站在“拿到到期结算”的角度看，这组区间是适合直接做、轻仓做，还是应该跳过。

挂单建议

站在“现在这个 Ask 要不要吃”的角度看，是否更适合挂更低价格等成交。

波段建议

站在短线价格波动角度看，是否值得先观察、等回落，或者只做交易价差。

建议状态怎么理解

直接建仓 / 小仓试单

说明模型判断和市场价格差得还可以，值得开始参与，但也不代表没有风险。

限价挂单 / 等待回落

说明方向可能没错，但当前卖价偏贵，直接追进去不划算，更适合等更好的成交价。

波段观察 / 跳过

说明更适合先看，不适合按当前价格做结算；如果 Edge 很差，通常直接跳过更稳。

盘口待刷新

不是这个区间一定没机会，而是当前这次刷新没有拿到可靠盘口，先别按这条数据下判断。

什么时候更要谨慎

  • 市场深度很薄，但你预算很大，容易出现滑点。
  • 组合成本已经明显高于命中概率，说明“买得太贵”。
  • 离结算时间很近，但 xtracker 计数还在快速变化，模型会更容易被短时波动打乱。
  • 你选了太多边缘区间，表面上覆盖更广，实际上很可能把成本堆高。

一个实用心法

不要把这个面板当成“自动赚钱按钮”，它更像是一个价格和概率的对照尺。你真正要做的，是找出 “市场暂时给错价” 的地方，而不是看到有概率就买。先看价格，再看概率，最后再决定要不要出手。
